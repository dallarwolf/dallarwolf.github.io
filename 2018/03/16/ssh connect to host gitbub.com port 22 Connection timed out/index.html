ssh connect to host gitbub.com port 22 Connection timed out

#win10通过SSH于GitHub上提交博客ssh connect to host gitbub.com port 22 Connection timed out

通过ssh向github上提交博客时出现ssh connect to host gitbub.com port 22 Connection timed out

方法：博客目录文件夹下将_config.yml文件中deploy下

repository: git@gitbub.com:username/username.github.io.git

更改为

repository: https://username@github.com/username/username.github.io.git

斜体下划线username部分更改为自己的github名称

原因：主要为SSH22端口某种原因不可用ssh协议导致 connect to host gitbub.com port 22 Connection timed out

把提交的url更改为https协议就可以使用了
